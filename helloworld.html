创建项目结构

打开终端,切换到所想建立项目的目录下,输入 (你也可以设置在当前目录下打开终端的快捷键)

稍等一段时间(视你的网络情况,将近10分钟都可能有的),等待终端完成 你会发现在当前目录下已经新建好一个叫"HelloWorld"的文件夹,里面结构如下:

先不用管里面的东西,后面会有详细的解释,现在我们的目标是尽快的让它跑起来.

跑起来!

  • 运行Xcode
  • 通过菜单栏-文件-导航到我们刚才创建的HelloWorld目录

  • 然后我们选择一个ios模拟器,这里我选了5s(穷)
  • command+R 快捷键:编译并运行

动图演示: 运行我们发现会弹出两个窗口

  • 终端窗口-显示React Package

在终端上会打开我们刚才零编写得到应用,欢迎界面上写着四个大字：欢迎入坑

天有不测风云,如果一样的神操作你最终得到这样的画面 十有八九是没开服务,打开终端切到项目目录运行npm start即可

下面为我们的小HelloWorld添点东西，尝试放一个文本输入框用来输入名字，当你输入"张三"时,下面的字就相应变为欢迎张三入坑。

来，我们描述一下要如何实现功能：

  • 添加一个UI组件，还要附给它一个样式,让我们能够输入文字,并且通过样式也知道那家伙是用来输入的
  • 添加一个处理输入文本的方法
  • 添加一段文本 用来输出我们输入的"问候语句"

按照这个思路,这个app应该会变成这样：

下面是更新后的代码清单:

/**
 * Sample React Native App
 * https://github.com/facebook/react-native
 */
'use strict';

var React = require('react-native');
var {
  AppRegistry,
  StyleSheet,
  Text,
  TextInput,
  View,
} = React;

var HelloWorld = React.createClass({
  getInitialState:function(){
    return {
      name:"World"
    };
  },
  onNameChanged:function(event){
    this.setState({name:event.nativeEvent.text});
  },
  render: function() {
    return (
      <View style={styles.container}>
        <TextInput
          style={styles.nameInput}
          onChange={this.onNameChanged}
          placeholder='输入要入坑宝宝的名字' />
        <Text style={styles.welcome}>
          欢迎{this.state.name}入坑
        </Text>
        <Text style={styles.instructions}>
          太骚气了
        </Text>
      </View>
    );
  }
});

var styles = StyleSheet.create({
  container: {
    flex: 1,
    justifyContent: 'center',
    alignItems: 'center',
    backgroundColor: '#F5FCFF',
  },
  welcome: {
    fontSize: 20,
    textAlign: 'center',
    margin: 10,
  },
  instructions: {
    textAlign: 'center',
    color: '#333333',
    marginBottom: 5,
  },
  nameInput:{
    height:36,
    padding:4,
    margin:24,
    fontSize:18,
    borderWidth:1
  }
});

AppRegistry.registerComponent('HelloWorld', () => HelloWorld);

借着说这个,介绍一个新的React-Native的UI元素——TextInput:

  • 引入TextInput 第12行
  • 在第28行我们把它添加到了界面上
  • 在63行是附加给它的样式
  • 在17行设置了一个初始内容
  • 我们还定了一个叫onNameChanged的方法,它的作用就是改变文本(第22行)
  • 在第30行把onNameChange方法绑定到输入框的onChange事件
  • 在第33行用插值表达式来输出文本，该文本动态且根据输入框输入的内容进行显示

重新运行这个应用,会发现当前界面多出了一个输入框,正如上面所描述的，输入框和输出内容是动态同步显示的，你可以理解为一个双工绑定.
